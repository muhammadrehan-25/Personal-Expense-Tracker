# Personal Expense Tracker

A simple and efficient Python-based application to manage and track daily expenses.

## ✨ Features
* **Add Expenses:** Input your daily spending with ease.
* **View History:** See a list of all recorded expenses.
* **Data Persistence:** Automatically saves all data to a JSON file so you don't lose your records.

## 🛠️ Built With
* **Python**: Core logic and functionality.
* **JSON**: Used for lightweight data storage.

## 🚀 How to Run
1. Clone this repository.
2. Run the Python script:
   ```bash
   python expense_tracker.py
